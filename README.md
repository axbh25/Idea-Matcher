# Event Matchmaker

Event Matchmaker is a beginner-friendly hackathon MVP that recommends project ideas or teammate roles based on a user's interests, skills, and preferred hackathon theme.

## Live Demo

idea-matcher--axbh25.replit.app

## Problem

Hackathon beginners often struggle to quickly choose a realistic project idea or understand what role they should take on a team.

## Solution

Event Matchmaker asks for a user's interests, skill level, and preferred theme, then generates 3 project or role recommendations that are simple enough to build during a hackathon.

## Features

- User profile form
- Interest and skill input
- Theme-based recommendations
- 3 project or role matches
- Suggested tech stack
- 30-second pitch idea
- Copy results button
- Reset button
- Mobile-responsive layout
