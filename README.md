# Event Matchmaker

Event Matchmaker is a beginner-friendly hackathon MVP that recommends project ideas or teammate roles based on a user's interests, skills, and preferred hackathon theme.

## Live Demo

https://idea-matcher--axbh25.replit.app/

## Problem

Hackathon beginners often struggle to quickly choose a realistic project idea or understand what role they should take on a team. This can waste valuable time at the beginning of a hackathon and make it harder to start building.

## Solution

Event Matchmaker asks users for their interests, skill level, and preferred hackathon theme, then generates three project or role recommendations. Each recommendation includes a project idea, problem statement, suggested role, difficulty level, tech stack, and short pitch idea.

## Features

- User profile form
- Interest and skill input
- Theme-based recommendations
- 3 project or role matches
- Suggested role for the user
- Suggested tech stack
- Difficulty level
- 30-second pitch idea
- Copy results button
- Reset button
- Mobile-responsive layout

## Tech Stack

- Replit
- Replit Agent
- React-style frontend
- Mock recommendation logic
- GitHub for portfolio documentation
